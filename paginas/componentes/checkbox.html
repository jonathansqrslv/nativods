Checkbox / Radio / Switch

Controles de seleção. Checkbox para múltipla escolha, radio para escolha única, switch para ativar/desativar.

Checkbox

Quatro estados possíveis: padrão, marcado, desabilitado e marcado+desabilitado.

<div class="checkbox">
  <input class="checkbox__input" type="checkbox">
  <label class="checkbox__label">Lembrar de mim</label>
</div>
<div class="checkbox">
  <input class="checkbox__input" type="checkbox" checked>
  <label class="checkbox__label">Aceito os termos de uso</label>
</div>
<div class="checkbox">
  <input class="checkbox__input" type="checkbox" disabled>
  <label class="checkbox__label">Opção indisponível</label>
</div>
<div class="checkbox">
  <input class="checkbox__input" type="checkbox" checked disabled>
  <label class="checkbox__label">Marcado e desabilitado</label>
</div>

Checkbox inline

Com .checkbox-group--inline, os itens ficam lado a lado.

<div class="checkbox-group checkbox-group--inline">
  <div class="checkbox">
    <input class="checkbox__input" type="checkbox" checked>
    <label class="checkbox__label">Design</label>
  </div>
  <div class="checkbox">
    <input class="checkbox__input" type="checkbox">
    <label class="checkbox__label">Código</label>
  </div>
  <div class="checkbox">
    <input class="checkbox__input" type="checkbox" checked>
    <label class="checkbox__label">Pesquisa</label>
  </div>
</div>

  <label class="switch__label">Notificações ativas</label>
</div>
