Input

Campos de entrada para formulários. Estilo underline com feedback visual claro.

Básico

Inputs com rótulo, placeholder e dica em um bloco .field.

Deve conter letras e números. 
<div class="field">
  <label class="field__label" for="inp-nome">Nome completo</label>
  <input type="text" id="inp-nome" class="input" placeholder="Seu nome">
</div>

<div class="field">
  <label class="field__label" for="inp-email">E-mail</label>
  <input type="email" id="inp-email" class="input" placeholder="ana@email.com">
</div>

<div class="field">
  <label class="field__label" for="inp-senha">Senha</label>
  <input type="password" id="inp-senha" class="input" placeholder="Mínimo 8 caracteres">
  <span class="field__hint">Deve conter letras e números.</span>
</div>

Tamanhos

Três densidades: .input--sm, padrão e .input--lg.

<input class="input input--sm" placeholder="Input sm">
<input class="input" placeholder="Input padrão">
<input class="input input--lg" placeholder="Input lg">

Estados

Erro, sucesso, desabilitado e somente leitura.

Informe um e-mail válido. 
<input class="input input--error" value="email-invalido">
<span class="field__error">Informe um e-mail válido.</span>

<input class="input input--success" value="Ana Silva">

<input class="input" value="Não editável" disabled>

<input class="input" value="Apenas leitura" readonly>

Com ícone

Envolva o input em .input-wrapper com ícones antes ou depois.

<div class="input-wrapper">
  <svg class="input-addon" ...><!-- ícone de busca --></svg>
  <input type="text" class="input" placeholder="Pesquisar...">
</div>

<div class="input-wrapper">
  <input type="password" class="input" placeholder="Digite sua senha">
  <button class="input-addon" aria-label="Mostrar senha">
    <svg ...><!-- ícone olho --></svg>
  </button>
</div>

Textarea

A classe .input também funciona em textareas.

Máximo 500 caracteres. 
<div class="field">
  <label class="field__label" for="inp-msg">Mensagem</label>
  <textarea id="inp-msg" class="input" rows="4" placeholder="Escreva sua mensagem..."></textarea>
  <span class="field__hint">Máximo 500 caracteres.</span>
</div>

Select

A classe .input também se aplica a selects nativos.

<div class="field">
  <label class="field__label" for="inp-select">Estado</label>
  <select id="inp-select" class="input">
    <option value="" disabled selected>Selecione um estado</option>
    <option value="sp">São Paulo</option>
    <option value="rj">Rio de Janeiro</option>
    <option value="mg">Minas Gerais</option>
    <option value="ba">Bahia</option>
  </select>
</div>

Addon

Combine input com botões de ação adjacentes.

<div class="input-wrapper">
  <input type="text" class="input" placeholder="https://">
  <button class="input-addon btn btn--solid btn--sm">Verificar</button>
</div>
