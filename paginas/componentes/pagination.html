Paginação

Navegação entre páginas de resultados com controles de avanço e retrocesso.

Padrão

Paginação completa com setas de navegação, números de página e elipse para intervalos extensos.

… 
<nav class="pagination">
  <button class="pagination__item" aria-label="Anterior">
    <svg ...><polyline points="15 18 9 12 15 6"/></svg>
  </button>
  <button class="pagination__item pagination__item--active" aria-current="page">1</button>
  <button class="pagination__item">2</button>
  <button class="pagination__item">3</button>
  <span class="pagination__ellipsis">&hellip;</span>
  <button class="pagination__item">8</button>
  <button class="pagination__item">9</button>
  <button class="pagination__item" aria-label="Próximo">
    <svg ...><polyline points="9 18 15 12 9 6"/></svg>
  </button>
</nav>

Outline

Use pagination--outline para uma variante com bordas visíveis em todos os itens. O item ativo recebe borda colorida sem preenchimento.

… 
<nav class="pagination pagination--outline">
  <button class="pagination__item" aria-label="Anterior">
    <svg ...></svg>
  </button>
  <button class="pagination__item pagination__item--active" aria-current="page">1</button>
  <button class="pagination__item">2</button>
  ...
</nav>

Simples

Use pagination--simple para uma navegação reduzida com apenas botões de avançar/retroceder e informação da página atual.

Página 3 de 12 
<nav class="pagination pagination--simple">
  <button class="pagination__item" aria-label="Anterior">
    <svg ...></svg>
  </button>
  <span class="pagination__info">Página 3 de 12</span>
  <button class="pagination__item" aria-label="Próximo">
    <svg ...></svg>
  </button>
</nav>

Tamanhos

Use pagination--sm para contextos compactos e pagination--lg para interfaces que pedem mais destaque.

pagination--sm
pagination--lg
<nav class="pagination pagination--sm">
  <button class="pagination__item" ...></button>
  <button class="pagination__item pagination__item--active" ...>1</button>
  ...
</nav>

<nav class="pagination pagination--lg">
