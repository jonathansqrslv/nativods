Layout

Primitivos de layout para estruturar páginas com grid, stack e containers responsivos.

Container

Larguras máximas responsivas para centralizar conteúdo. Use .container com modificadores container--sm, container--lg ou container--xl.

Container MD (padrão)

<div class="container container--xl">...</div>
