Dropdown

Menu suspenso para ações contextuais, navegação ou seleção.

Básico

Menu com itens de ação e separador. Use dropdown__item--danger para ações destrutivas.

<div class="dropdown">
  <button class="btn btn--neutral btn--sm dropdown__trigger">Ações</button>
  <div class="dropdown__menu">
    <button class="dropdown__item">Editar</button>
    <button class="dropdown__item">Duplicar</button>
    <button class="dropdown__item">Arquivar</button>
    <div class="dropdown__separator"></div>
    <button class="dropdown__item dropdown__item--danger">Excluir</button>
  </div>
</div>

Com ícones

Itens podem incluir ícones SVG para reforçar o significado da ação.

<div class="dropdown">
  <button class="btn btn--neutral btn--sm dropdown__trigger">Projeto</button>
  <div class="dropdown__menu">
    <button class="dropdown__item dropdown__item--active">
      <svg>...</svg> Editar
    </button>
    <button class="dropdown__item">
      <svg>...</svg> Duplicar
    </button>
    <button class="dropdown__item">
      <svg>...</svg> Arquivar
    </button>
    <div class="dropdown__separator"></div>
    <button class="dropdown__item dropdown__item--danger">
      <svg>...</svg> Excluir
    </button>
  </div>
</div>

Seções

Use dropdown__label para agrupar itens com um rótulo descritivo.

Equipe
Canais
<div class="dropdown">
  <button class="btn btn--neutral btn--sm dropdown__trigger">Compartilhar</button>
  <div class="dropdown__menu">
    <div class="dropdown__label">Equipe</div>
    <button class="dropdown__item">Ana Silva</button>
    <button class="dropdown__item">Carlos Mendes</button>
    <div class="dropdown__separator"></div>
    <div class="dropdown__label">Canais</div>
    <button class="dropdown__item">Copiar link</button>
    <button class="dropdown__item">Enviar por e-mail</button>
  </div>
</div>
