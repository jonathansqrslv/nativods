Alert

Mensagem de feedback para informar, alertar ou confirmar ações do usuário.

Variantes

Quatro variantes para cada contexto de feedback: informação, sucesso, atenção e erro.

Informação
Uma atualização está disponível para o seu sistema.
Sucesso
Seu documento foi salvo com sucesso.
Atenção
Seu plano expira em 3 dias. Renove para continuar.
Erro
Não foi possível processar o pagamento. Tente novamente.

  <div class="alert__icon"></div>
  <div class="alert__content">
    <div class="alert__title">Informação</div>
    <div class="alert__body">Uma atualização está disponível.</div>
  </div>
  <button class="alert__close" aria-label="Fechar"><svg>...</svg></button>
</div>

Com ícones

Alertas com ícones personalizados na área lateral para reforçar o significado.

Dica
Use atalhos de teclado para navegar mais rápido.
Concluído
Todos os itens foram sincronizados com o servidor.

Acento lateral

Variante com borda lateral colorida, sem borda completa — mais discreta para uso inline.

Novidade
Nova versão do painel disponível com melhorias de performance.
Verificado
Sua conta foi verificada com sucesso. Bem-vindo.
Falha
O upload do arquivo excede o limite de 10 MB.

Toast

Notificação flutuante de canto, com botão de fechar. O posicionamento e a animação são controlados via JavaScript.

Salvo
