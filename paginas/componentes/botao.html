Botão

O botão é o elemento acionador de ações. Use para enviar formulários, confirmar decisões ou navegar.

Variantes

Seis variantes para cada contexto: ações primárias, secundárias e destrutivas.

<button class="btn btn--solid">Solid</button>
<button class="btn btn--outline">Outline</button>
<button class="btn btn--ghost">Ghost</button>
<button class="btn btn--neutral">Neutral</button>
<button class="btn btn--danger">Danger</button>
<button class="btn btn--danger-outline">Danger Outline</button>

Tamanhos

De xs a xl, para qualquer densidade de interface.

<button class="btn btn--solid btn--xs">XS</button>
<button class="btn btn--solid btn--sm">SM</button>
<button class="btn btn--solid">MD</button>
<button class="btn btn--solid btn--lg">LG</button>
<button class="btn btn--solid btn--xl">XL</button>

Ícone

Botões só de ícone adaptam o tamanho automaticamente.

<button class="btn btn--solid btn--icon btn--sm" aria-label="Voltar">
  <svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round"><polyline points="15 18 9 12 15 6"/></svg>
</button>

Estados

Estados desabilitado e carregando.

<button class="btn btn--solid" disabled>Desabilitado</button>
<button class="btn btn--outline" disabled>Desabilitado</button>
<button class="btn btn--solid btn--loading" disabled>Carregando</button>

Bloco

Botões que ocupam toda a largura disponível.

<button class="btn btn--solid btn--block">Botão de bloco</button>
<button class="btn btn--outline btn--block">Outline de bloco</button>
