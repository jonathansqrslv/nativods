Fundação

Tokens de design que formam a base visual da Brisa — cores, tipografia, espaçamento, sombras e movimento. Tudo conectado por variáveis CSS.

Cores

Tipografia

Escala tipográfica Major Second (~1.25). A voz do texto vai do micro (--text-2xs) ao monumental (--text-6xl).

Tamanhos

--text-6xlTítulo amplo 60px
--text-5xlTítulo de abertura 48px
--text-4xlTítulo de seção 36px
--text-3xlTítulo interno 30px
--text-2xlChamada curta 24px
--text-xlSubtítulo 20px
--text-lgTexto grande — leitura breve 18px
--text-baseTexto base — leitura contínua, sem pressa. 16px
--text-smTexto menor — apoio, legendas e rótulos. 14px
--text-xsTexto mínimo — metadados e contexto. 12px
--text-2xsTexto micro — raro, só onde essencial. 10px
/* Escala Tipográfica */
--text-6xl:  3.75rem;  /* 60px */
--text-5xl:  3rem;     /* 48px */
--text-4xl:  2.25rem;  /* 36px */
--text-3xl:  1.875rem; /* 30px */
--text-2xl:  1.5rem;   /* 24px */
--text-xl:   1.25rem;  /* 20px */
--text-lg:   1.125rem; /* 18px */
--text-base: 1rem;     /* 16px */
--text-sm:   0.875rem; /* 14px */
--text-xs:   0.75rem;  /* 12px */
--text-2xs:  0.625rem; /* 10px */

/* Uso */
<p style="font-size: var(--text-xl)">Subtítulo</p>
<p style="font-size: var(--text-base)">Corpo</p>
<span style="font-size: var(--text-sm)">Apoio</span>

Sombras

Sombras sutis, sem exagero. Cada nível eleva o elemento com a intensidade certa. O modo escuro ajusta a opacidade automaticamente.

--shadow-xs
--shadow-sm
--shadow-md
--shadow-lg
--shadow-xl
--shadow-2xl
--shadow-inner
--shadow-focus 
/* Sombras */
--shadow-xs:     0 1px 2px rgba(0,0,0,0.04);
--shadow-sm:     0 1px 3px rgba(0,0,0,0.05);
--shadow-md:     0 2px 6px rgba(0,0,0,0.06);
--shadow-lg:     0 4px 12px rgba(0,0,0,0.07);
--shadow-xl:     0 8px 20px rgba(0,0,0,0.08);
--shadow-2xl:    0 12px 32px rgba(0,0,0,0.1);
--shadow-inner:  inset 0 2px 4px rgba(0,0,0,0.04);
--shadow-focus:  0 0 0 2px var(--interactive-default);

/* Uso */
<div class="card card--elevated">...</div>
<input class="input" /* focus: var(--shadow-focus) */>

Movimento

Animações contidas, com propósito. Durações curtas e curvas de easing suaves. Tudo desativado quando o usuário prefere redução de movimento.

Durações

TokenValorUso
--duration-instant0msMudanças instantâneas, sem transição
--duration-fast100msMicro-interações, hover, foco
--duration-normal200msTransições padrão de cor e fundo
--duration-slow300msEntrada de modais e drawers
--duration-slower500msTransições de página, raras

Easing

TokenCurvaUso
--ease-linearlinearSpinners, loaders contínuos
--ease-incubic-bezier(0.4, 0, 1, 1)Elementos saindo da tela
--ease-outcubic-bezier(0, 0, 0.2, 1)Elementos entrando na tela
--ease-in-outcubic-bezier(0.4, 0, 0.2, 1)Transições simétricas
--ease-springcubic-bezier(0.34, 1.56, 0.64, 1)Movimento com retorno elástico

Transições compostas

TokenValorUso
--transition-fastall 100ms ease-outHover e foco rápidos
--transition-baseall 200ms ease-outTransição padrão de componentes
--transition-slowall 300ms ease-outEntrada de overlays
--transition-colorscolor, bg, border, shadow 200msMudança de cor sem mover layout

/* Transição de cor (usa --transition-colors) */

/* Respeita preferência de redução de movimento */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
