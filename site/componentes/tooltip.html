Tooltip

Balão informativo que aparece ao passar o mouse sobre um elemento.

Posições

Use .tooltip-wrapper no elemento pai e .tooltip com um modificador de posição no balão.

Dica no topo .tooltip--top
.tooltip--bottom Dica embaixo
Esquerda --left
--right Direita 
<span class="tooltip-wrapper">
  <button class="btn btn--ghost btn--sm">Passe o mouse</button>
  <span class="tooltip tooltip--top">Dica no topo</span>
</span>

<span class="tooltip-wrapper">
  <button class="btn btn--ghost btn--sm">Passe o mouse</button>
  <span class="tooltip tooltip--bottom">Dica embaixo</span>
</span>

<span class="tooltip-wrapper">
  <button class="btn btn--ghost btn--sm">Passe o mouse</button>
  <span class="tooltip tooltip--left">Esquerda</span>
</span>

<span class="tooltip-wrapper">
  <button class="btn btn--ghost btn--sm">Passe o mouse</button>
  <span class="tooltip tooltip--right">Direita</span>
</span>

Popover

Use .popover para um tooltip mais rico, com título e corpo.

Informação adicional
Um popover pode conter texto mais longo e elementos formatados.
<div class="popover">
  <div class="popover__title">Informação adicional</div>
  <div class="popover__body">Um popover pode conter texto mais longo.</div>
</div>
